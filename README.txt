# JotLink ‚Äì Notes Sharing App

JotLink is a Windows-focused notes application built with .NET MAUI and ASP.NET Core.  
It allows users to create, update, and manage notes, share them via unique links, and work offline using local TXT file storage.  

---

## ‚ú® Features
- Create, save, and update notes with a clean .NET MAUI UI  
- Share notes online via unique ASP.NET-generated links  
- Offline mode with local TXT file storage for persistence  
- Backend API hosted on Render
